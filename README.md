# Admission Form Webpage

A simple admission form webpage developed using HTML. This project is designed to collect basic student information through a structured and easy-to-use form.

## Features
- User-friendly form layout
- Fields for personal and academic details
- Input sections for name, email, gender, and other information
- Simple and organized design

## Technology Used
- HTML5

## Purpose
This project was created to practice HTML forms and understand different input elements used in web development.
